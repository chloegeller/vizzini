## Here is a brief description of each scenario:

<div className="py-2 w-100" />

A <span className="probable">**probable**</span> scenario is one that is
_believable_, and is _likely_ to happen.

An <span className="improbable">**improbable**</span> scenario is one that
is _**less** believable_, is _unlikely_ to happen, but **could** still happen.

An <span className="impossible">**impossible**</span> scenario is one that
is _**not** believable_, and **could not** happen.

A <span className="typeError">**type error**</span> scenario is one that
**could not** happen, because it **does not**
have proper _grammar_, _syntax_, or _semantics_.
